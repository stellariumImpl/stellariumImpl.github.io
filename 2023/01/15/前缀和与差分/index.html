前缀和与差分

一维差分

差分数组的前缀和等于原数组

arr 1 3 7 5 2
d 1 2 4 -2 -3
sum_d 1 3 7 5 2

为何如此标记差分数组

[L,R]+val<=>d[L]+val,d[R+1]−val[L,R]+val <=> d[L]+val,d[R+1]-val [L,R]+val<=>d[L]+val,d[R+1]−val

以[1,3]+1为例

d 0 1 0 0 0
sum_d 0 1 1 1 1

如果不做点什么，前缀和表观上的确受到了影响，4位置变成了1，所以通过差分数组标记对应位置-1，抵消这个影响。

d 0 1 0 0 -1
sum_d 0 1 1 1 0

二维前缀和

sum[i][j]=g[i][j]+sum[i][j−1]+sum[i−1][j]−sum[i−1][j−1]sum[i][j]=g[i][j]+sum[i][j-1]+sum[i-1][j]-sum[i-1][j-1] sum[i][j]=g[i][j]+sum[i][j−1]+sum[i−1][j]−sum[i−1][j−1]

sum[x1,y1][x2,y2]sum[x1,y1][x2,y2] sum[x1,y1][x2,y2]

=sum[x2][y2]−sum[x2][y1−1]−sum[x1−1][y2]+sum[x1−1][y1−1]=sum[x2][y2]-sum[x2][y1-1]-sum[x1-1][y2]+sum[x1-1][y1-1] =sum[x2][y2]−sum[x2][y1−1]−sum[x1−1][y2]+sum[x1−1][y1−1]
