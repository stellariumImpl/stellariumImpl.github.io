Recap

The greedy technique does not guarantee the best solution, but it does provide a good estimate in most circumstances. For some situations (such as Prim’s and Kruskal’s algorithms), greedy always produces the best answer. If greedy gives the best solution, it is the finest technique among all others.

Problems which can be solved using the greedy method generally possess the following two properties:

  • Greedy choice property: The global optimal solution is found by selecting locally optimal choices, or the ones that appear to be the best at the time. If the choice is feasible, include it in the solution set and reduce the problem by the same amount. The current decision may be influenced by prior decisions, but it is independent by future decisions.

  • Optimal substructure: We say given problem exhibits optimal substructure if the optimal solution to the given problem contains the optimal solution to its subproblems too. In the problem which possesses the optimal substructure, best next choice always leads to an optimal solution.

Greedy algorithms are used to find an optimal or near-optimal solution to many real-life problems. A few of them are listed as follow.

Minimum Spanning Tree

Tree = Connected acyclic graph

Spanning tree of graph G = subset of edges of G that form a tree & hit all vertices of G

MST: given a weighted graph G=(V,E) find a spanning tree T of minimum weight.

Greedy choice property: The greedy choice property refers to choosing the decision that is optimal in the current state at each step, without considering how future decisions will affect the final result. In Prim’s algorithm, the edge with the smallest weight connected to the set U is selected each time and the connected vertices are added to the set U. In Kruskal’s algorithm, the least weighted edge connecting two different connected components is chosen each time until all the vertices are in the same connected component. These greedy choices are based on the minimum weight edge in the current state, without considering the global optimal solution.

Optimal substructure: Optimal substructure means that the optimal solution of a problem can be constructed by a series of optimal solutions of subproblems. In Prim’s algorithm, each vertex is added to the minimum spanning tree as a subproblem, and their optimal solutions are combined to form the global optimal solution. In Kruskal’s algorithm, each selection of the edge with the minimum weight is also a subproblem, and each selection is made to construct the optimal solution for the entire minimum spanning tree.
