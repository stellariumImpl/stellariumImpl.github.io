笛卡尔积 from

sql中怎么比较字符串

用CAST()函数比较

语法：ORDER BY CAST(‘123’ AS SIGNED);

示例：

SELECT CAST(‘123’ AS SIGNED)>127; – 结果为0，表示false

SELECT CAST(‘123’ AS SIGNED)>12; – 结果为1，表示true

3.使用CONVERT()函数比较

语法：ORDER BY CONVERT(‘123’,SIGNED);
